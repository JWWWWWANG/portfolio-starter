The 20-minute flow

Five steps.

1 · Copy template → 2 · Enable Pages → 3 · Fill markdown

4 · Feed Gemini → 5 · Paste & publish → 6 · Iterate

The walk-through doc has every step in detail — keep it open on your laptop.

github.com/JWWWWWANG/portfolio-starter → WALKTHROUGH.md

Steps 1–2 · 2 minutes

Get your repo + turn on Pages.

1 · Copy the template

  • Open portfolio-starter
  • Click Use this template
  • Name it portfolio
  • Set to Public

2 · Enable Pages

  • Settings → Pages
  • Branch: main
  • Click Save
  • Write down your URL
[ SCREENSHOT: "Use this template" button + Pages settings page ]

Step 3 · 5 minutes

Fill the template.

Pick the template matching your discipline:

  • template-general.md — universal base
  • template-actor.md — actors
  • template-filmmaker.md — filmmakers
  • template-musician.md — musicians

Replace every [[placeholder]] with your content.

Short on time? Just fill: Hero + About + one project. Add the rest later.

Step 4 · 3 minutes · ⭐ The core moment

Feed Gemini — in order.

  • 4.1 — Drag your style reference screenshot into Gemini.
    "Remember this visual style — colors, typography, layout, vibe."
  • 4.2 — Paste the system prompt (from the walk-through doc).
  • 4.3 — Paste your filled markdown template.
  • 4.4 — Gemini outputs HTML. Cmd+A → Cmd+C.

⚠️ If the output starts with ` ```html ` — delete that line and the closing ` ``` `. Raw HTML only.

Step 5 · 3 minutes

Paste → commit → refresh.

  • In your repo, click index.html
  • Click the pencil icon (Edit this file)
  • Cmd+A → Delete → Paste the Gemini HTML
  • Scroll down → Commit changes
  • Open your Pages URL → refresh
[ SCREENSHOT: pencil icon + Commit changes button ]

🎉 Your site is live.

Step 6 · 3 minutes · The fun part

Direct Gemini with your voice.

Back in Gemini, try:

"Make the hero darker with a subtle gradient."
"Use Playfair Display for the headings."
"Add a fade-in animation when scrolling."
"Make it more brutalist / Y2K / editorial."
"Make it fully responsive on mobile."
"Add a dark mode toggle."

Copy → paste into index.html → commit → refresh.
Repeat until you love it.

If you get stuck

Plan B: Netlify Drop

No GitHub account? GitHub stuck? 30-second backup:

  • Open netlify.com/drop
  • Save Gemini's HTML as index.html on your desktop
  • Drag that file into the browser window
  • Netlify gives you a live URL immediately

Trade-off: random URL, manual updates. But it always works.

Other issues? Check the Troubleshooting section of the walk-through doc → WALKTHROUGH.md
